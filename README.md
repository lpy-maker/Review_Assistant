# 审核意见助手

> Chrome 侧边栏插件 — 模块化文本模板管理，一键组合输出，内置待办清单。

## 这是什么

一个通用的 **Chrome 侧边栏文本组装工具**。你可以按类别和模块预先录入常用文本片段，点击即可插入到输出区组合成完整文本，一键复制到剪贴板。同时内置分模块的待办清单，帮助跟踪复查项。

**典型场景**：文档审核意见、客服回复模板、代码审查备注、邮件模板、病历/法律文书片段……任何需要「从预制片段快速拼装文本」的场景都适用。

## 功能

- **三层组织结构** — 类别（L1）→ 模块（L2）→ 文本片段（L3），层次清晰
- **一键插入** — 点击片段插入到光标位置，自动拼接
- **一键复制** — 组合完成后一键复制到剪贴板
- **待办清单** — 每个类别独立的待办列表，支持勾选和排序
- **拖拽排序** — 模块、片段、待办均支持拖拽调整顺序
- **可拖拽面板** — 待办区和文本区高度自由调节
- **数据可迁移** — 支持 JSON 导出/导入，数据备份无忧
- **自动备份** — 可开启自动导出，每次修改数据后自动下载备份文件

### 自动导出

设置页开启「自动导出」后，每次对数据做修改（增删改类别、模块、片段、待办），2 秒后自动下载备份文件。

| 项目 | 说明 |
|------|------|
| 文件位置 | Chrome 默认下载文件夹 |
| 文件名 | `审核意见助手数据_自动备份.json` |
| 覆盖策略 | 每次覆盖同名文件，不产生冗余副本 |
| 建议用法 | 将该文件放入同步盘（OneDrive / 坚果云）或 Git 仓库，实现跨设备数据同步 |

关闭开关后停止自动导出，手动导出仍可随时使用。

## 安装

1. 克隆仓库或下载 ZIP 解压
   ```bash
   git clone https://github.com/lpy-maker/Review_Assistant.git
   ```
2. 打开 Chrome，进入 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」，选择项目文件夹
5. 点击浏览器工具栏的扩展图标即可打开侧边栏

## 使用

### 日常使用（侧边栏）

1. 顶部下拉选择 **类别**（如「项目验收」）
2. 勾选待办清单中的复查项
3. 点击横向标签切换 **模块**（如「汇总表」）
4. **点击文本片段** → 自动插入到下方输出区光标位置
5. 点击 **复制** → 粘贴到目标位置
6. 点击 **清空** 重置输出区和待办勾选状态

### 管理内容（设置页）

点击侧边栏右上角齿轮图标进入设置页：
- 左侧面板管理类别、待办、模块
- 右侧面板编辑具体内容
- 支持拖拽排序、双击文字直接编辑

## 数据结构

```
类别（L1）
├── 待办（L2'） — 该类别下的复查清单
└── 模块（L2）
    └── 文本片段（L3） — 点击插入的预制文本
```
